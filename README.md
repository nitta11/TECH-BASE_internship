# TECH-BASE_internship
## 概要
TECH-BASEのインターンでの制作物です。
PHPとMySQLを使った掲示板です。
## 機能
1.送信機能
2.削除機能
3.編集機能

編集、削除はどちらも送信時に設定したパスワードを入力することで実行できます。
